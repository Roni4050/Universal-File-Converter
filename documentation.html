Universal File Converter Documentation

This simple web app lets you convert JPG images to PDF files and PDF files to JPG images directly in your browser.

How to Use

  1. Open index.html in any modern browser (Chrome, Firefox, Edge, Safari).
  2. Select the conversion mode:
    • JPG to PDF: Upload a JPG image file and convert it to PDF.
    • PDF to JPG: Upload a PDF document and convert the first page to JPG.
  3. Click the file upload button and select your file.
  4. Click the convert button.
  5. Wait for processing, then download the converted file using the link provided.

Notes

  • All processing happens locally on your device, no files are uploaded online.
  • PDF to JPG converts only the first page of the PDF.
  • Supported input files:
    • JPG/JPEG images for JPG to PDF.
    • PDF documents for PDF to JPG.
  • If you encounter any errors, try with different files or ensure your files are not corrupted.

Assets

All external libraries are loaded from CDN, so no local assets are required.
