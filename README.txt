Universal File Converter
========================

This project is a simple web-based tool to convert JPG images to PDF documents and PDF documents to JPG images.

Features:
- Upload JPG images and convert them into a PDF file.
- Upload PDF documents and convert the first page into a JPG image.
- Client-side processing using modern JavaScript libraries.
- Responsive and modern UI using Tailwind CSS.
- No backend required.

Usage:
1. Open index.html in any modern browser.
2. Select conversion mode (JPG to PDF or PDF to JPG).
3. Upload the appropriate file.
4. Click the convert button.
5. Download the converted file.

All processing happens in the browser, so files are never uploaded anywhere.
